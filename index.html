Above is a responsive navigation bar. Each list item is a link to a certain part of the page, complete with hover states. When the window decreses in size, the links are hidden until the "Show More" button is pressed, resulting in a vertical block display. The "Show less" button will hide the button, and the li's will return to their initial state when the window is returned to non-mobile size.
